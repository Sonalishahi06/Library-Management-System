# 📚 Library Management System

## 🚀 Tech Stack
- Java (Swing)
- MySQL
- NetBeans

## 📌 Features
- Add / Remove Books
- Manage Staff
- Admin Login System
- Update Credentials

## 💡 Description
A desktop-based Library Management System built using Java and MySQL to manage books and staff efficiently.
